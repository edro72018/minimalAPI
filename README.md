# MyWebApp - minimalAPI en .NET 8

## Descripción
Minimal API para gestionar tareas (to do), usando .NET 8.


## Requisitos
- .NET 8 SDK instalado
- VS Code
    - Extensiones: REST Client de Huachao Mao, para procesar las peticiones HTTP
- actualmente usa almacenamiento en memoria

## Cómo ejecutar
en terminal:
dotnet restore
dotnet run
